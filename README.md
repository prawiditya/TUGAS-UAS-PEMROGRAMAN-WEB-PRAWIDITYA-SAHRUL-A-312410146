RUD artikel (detail, hapus, tambah, ubah).
    * `dashboard/`: Halaman panel kontrol pengguna.
    * `home/`: Halaman utama publik.
    * `profile/`: Pengaturan profil dan data diri (edit, update).
    * `settings/`: Pengaturan sistem aplikasi.
    * `users/`: Manajemen data pengguna (edit, hapus, tambah).
* **`/template`**: Berisi potongan kode UI (partial views) untuk konsistensi tampilan:
    * `header.php`: Bagian atas halaman (navigasi).
    * `footer.php`: Bagian bawah halaman.
    * `sidebar.php`: Menu navigasi samping.
* **`.htaccess`**: Konfigurasi server Apache untuk manajemen URL rewriting (clean URL).
* **`config.php`**: Konfigurasi utama aplikasi termasuk koneksi database dan konstanta global.
* **`index.php`**: Entry point utama aplikasi yang menangani routing antar modul.
